# ギターコード練習アプリ

咲季ちゃんのギター練習をサポートするシンプルなウェブアプリです。

## 機能

- **14種類の基本コード** - メジャー、マイナー、セブンスコードを収録
- **大きなコードダイアグラム** - 押さえる位置が一目でわかる
- **指番号表示** - どの指で押さえるかを色分けで表示
- **モバイル対応** - iPad miniでも快適に使える

## 収録コード

### メジャーコード
C, D, E, F, G, A

### マイナーコード
Am, Dm, Em, Bm

### セブンスコード
C7, D7, G7, A7

## 使い方

1. コードボタンをタップ
2. ダイアグラムでフレットと指の位置を確認
3. ギターで押さえて練習！

## 指番号の見方

| 色 | 指 |
|---|---|
| 赤 | 人差し指 (1) |
| 緑 | 中指 (2) |
| 青 | 薬指 (3) |
| 紫 | 小指 (4) |

## 技術スタック

- HTML5
- CSS3 (レスポンシブデザイン)
- JavaScript (バニラJS)
- SVGでコードダイアグラムを動的生成
