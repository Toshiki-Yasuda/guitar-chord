# ギターコード練習アプリ

咲季ちゃんのギター練習をサポートするシンプルなウェブアプリです。

## 機能

- **14種類の基本コード** - メジャー、マイナー、セブンスコードを収録
- **大きなコードダイアグラム** - 押さえる位置が一目でわかる
- **指番号表示** - どの指で押さえるかを色分けで表示
- **10種類のアルペジオパターン** - 上昇、下降、往復、ピンチ、3本指、4本指、トラビス、ロール、カスケード、ランダム
- **コード進行機能** - コードを登録して連続再生
- **モバイル対応** - iPad miniでも快適に使える
- **PWA対応** - ホーム画面に追加してアプリのように使える、オフラインでも動作

## 収録コード

### メジャーコード
C, D, E, F, G, A

### マイナーコード
Am, Dm, Em, Bm

### セブンスコード
C7, D7, G7, A7

## 使い方

1. コードボタンをタップ
2. ダイアグラムでフレットと指の位置を確認
3. ギターで押さえて練習！

## 指番号の見方

| 色 | 指 |
|---|---|
| 赤 | 人差し指 (1) |
| 緑 | 中指 (2) |
| 青 | 薬指 (3) |
| 紫 | 小指 (4) |

## 技術スタック

- HTML5
- CSS3 (レスポンシブデザイン)
- JavaScript (バニラJS)
- SVGでコードダイアグラムを動的生成
- PWA (Progressive Web App)
  - Service Worker でオフライン対応
  - Web App Manifest でホーム画面追加対応

## PWA として使用する

このアプリはPWA（Progressive Web App）として動作します:

1. スマートフォンの場合
   - ブラウザでアプリを開く
   - 「ホーム画面に追加」を選択
   - ホーム画面からアプリを起動

2. パソコンの場合
   - ブラウザでアプリを開く
   - アドレスバーの「インストール」ボタンをクリック

詳しくは [PWA-README.md](PWA-README.md) をご覧ください。
